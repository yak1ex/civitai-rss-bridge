
# civitai-rss-bridge

This is a bridge for Civitai Models for [RSS-Bridge](https://github.com/RSS-Bridge/rss-bridge).

## Install

Place `CivitaiBridge.php` at `bridges/` folder.
Note that currently `CACHE_TIMEOUT` is set as `0` for testing purpose.

If you use [tt-rss](https://github.com/tt-rss/tt-rss), you can restore intended image views by copying content of `tt-rss.css` into tt-rss custom CSS area.
